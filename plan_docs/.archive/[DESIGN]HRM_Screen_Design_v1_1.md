| KPI 카드 — 부분 가동 | Stat Card (yellow) | `HR_AVAIL_SNAP.AVAIL_STAT_CD = 'PARTIAL'` COUNT |
| KPI 카드 — 풀 투입 | Stat Card (red) | `HR_AVAIL_SNAP.AVAIL_STAT_CD = 'FULL'` COUNT |

| 부서별 가동률 | Horizontal Bar | `HR_DEPT_MST` JOIN `HR_AVAIL_SNAP` AVG |
| **조직 평균 가동률 3단계** | Progress Bar 3행 | ① `ASGN_TYPE_CD='RUNNING'` 만 집계 ② `+'COMMITTED'` 포함 ③ `+'PROPOSED'` 전체 |

>
> | 단계 | 포함 유형 | 색상 |
> |---|---|---|

>
> **사원 계정 자동 생성 (2026-07-06 설계 변경):** "신규 등록" 시 이메일이 필수 입력이 되며(§5.3.1 `HR_EMPL_MST.EMAIL_ADDR` NOT NULL), 등록 완료 즉시 `SYS_USER_MST` 계정이 자동 생성된다 — 상세 규칙은 설계 문서 §5.5 업무 규칙 "사원 계정 자동 생성" 및 SCR-015 참조. 등록 처리 결과 화면에 임시 비밀번호가 1회 노출되므로, 등록자가 이를 신규 사원에게 안전하게 전달해야 한다.
>
> 실제 팀 코드: **딜리버리 / 세일즈파트너 / 영업**
> 실제 직급 체계: **인턴 · 사원 · 대리 · 차장 · 부장 · 이사 · 상무보 · 상무 · 전무 · 부대표**
> 실제 보유역할 코드: **AA · TA · DBA · BA · PM · 컨설턴트 · 사업관리** (복수 가능, 쉼표 구분)
